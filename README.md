# UofT Data Science Institute - Coursework

My completed coursework from the University of Toronto Data Science Institute program.

## Program Structure

### 01 - Foundations
- **Shell**: Command line basics, bash scripting, file management
- **Git**: Version control, GitHub workflows, collaboration
- **Python**: Core programming, data structures, functions, OOP

### 02 - Data
- **SQL**: Database queries, joins, aggregations, optimization
- **Applying Statistical Concepts (LCR)**: Learning, Classification, Regression

### 03 - Algorithms  
- **Algorithms & Data Structures**: Complexity analysis, sorting, searching

### 04 - Advanced
- **Deep Learning**: Neural networks, TensorFlow, model training
- **Scaling to Production**: MLOps, deployment, monitoring
